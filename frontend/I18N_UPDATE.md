# I18n Synchronization & UI Localization Update

## 📋 Overview
Previously, several UI components contained hardcoded English strings that did not toggle when the language was switched to Vietnamese. This update identifies, extracts, and migrates those strings into the `next-intl` translation framework.

## 🚀 How to Add New Translations
1. **Identify hardcoded text** in any `.tsx` file.
2. **Add a key** to `frontend/messages/en.json` and `frontend/messages/vi.json`.
3. **Import the hook**:
   ```tsx
   import { useTranslations } from 'next-intl';
   // ... inside component:
   const t = useTranslations('category');
   return <div>{t('your_key')}</div>;
   ```
4. **For Server Components**, use `getTranslations` from `next-intl/server`.
